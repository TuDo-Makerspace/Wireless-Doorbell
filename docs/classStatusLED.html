Detailed Description

StatusLED class.

The StatusLED class provides a simple "asynchronous" interface for controlling status LEDs. LEDs can be set to blink, be on or off. In order to work "asynchronously", the class requires the user to continously the update() method.

◆ mode()

void StatusLED::mode ( led_mode  m)

Sets the LEDs mode.

The following function sets the LEDs mode. Available modes are:

  • OFF LED is off
  • ON LED is on
  • BLINK LED blinks ON, then OFF
  • BLINK_INV LED blinks OFF, then ON

The BLINK_INV mode is useful for alternating blinks between two LEDs.

Parameters
modeThe LEDs mode
